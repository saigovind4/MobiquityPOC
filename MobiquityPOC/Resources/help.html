How To Use The Weather App :


  1. The Home Screen

    1. The home screen shows the bookmarked cities.

    2. To Add a location/ city clock on the location button which would navigate to the map screen for adding the locaiton.

    3. After adding the city you may search through the bookmarked cities.

    4. For deleting the city from the list please swipe left by holding the cell.

  2. The MapView Screen

    1. The MapView Screen uses Apple Maps kit and buil-in MapsView layout.

    2. Select or tap or drag the pin across the map to select the city.

    3. The search bar on the top lets the user search the precise city based on the search keyword ex: London, Mumbai, Hyderabad.

    4. The current location button redirects the user to his/her location based on the coordinates given.

  3. Keeping The Data Secure With CoreData

    1. This app uses in-built CoreData framework for storing, retriving and managing the location data.

    2. If the user uninstalls the app, the data would be gone.

    3. Core Data abstracts the details of mapping your locations to a store, making it easy to save data from Swift.

    4. Core Data is one of the most popular frameworks provided by Apple for iOS and macOS apps.
